# 🐍 Snake Apple Game
A classic Snake game built using Pygame, enhanced with power-ups and obstacles for a more challenging and fun experience.

🎮 Features
🍎 Eat Apples: Grow longer by eating apples.

⚡ Power-ups:

🟡 Double: Doubles your score.

🔵 Shrink: Shrinks the snake length.

🟣 Speed: Temporarily increases snake speed.

⬛ Obstacles: Avoid obstacles as you move.

💾 High Score: Keeps track of your best score.

🎵 Background music and sound effects.

# **🚀 Getting Started
Prerequisites
 Make sure Python 3.x is installed on your system. Install Pygame:
 -->  pip install pygame

# **Installation
Clone this repository:
 -->  git clone https://github.com/yourusername/snake-apple-game.git
 -->  cd snake-apple-game
 
 **Run the game:**
   python main.py

   
# 🕹️ Controls: 
| Key    | Action     |
| ------ | ---------- |
| ⬆️ / W | Move Up    |
| ⬇️ / S | Move Down  |
| ⬅️ / A | Move Left  |
| ➡️ / D | Move Right |
| P      | Pause Game |
| Q      | Quit Game  |
